# Product Marketing Context

*Last updated: 2026-04-25*

## Product Overview
**One-liner:** 국민연금 국내주식 보유 변동 대시보드.

**What it does:** DART의 대량보유 공시에서 국민연금공단 관련 국내주식 보유 변동 이벤트를 수집, 파싱, 저장한다. 특정 기준일의 포트폴리오 스냅샷, 섹터 변화, 10% 이상 보유주 전략 정보를 HTML 대시보드와 CSV로 산출한다. 결과물은 GitHub Pages에 정적 파일로 배포한다.

**Product category:** 국내주식 데이터 대시보드 / 국민연금 보유종목 추적기 / DART 공시 기반 투자 리서치 도구.

**Product type:** 데이터 분석 프로젝트, 정적 웹 대시보드, 로컬 CLI 도구.

**Business model:** 현재는 개인 리서치용 무료 프로젝트. 향후 투자 콘텐츠, 리포트, 뉴스레터, 유료 데이터 서비스 등으로 확장 가능성이 있음.

## Target Audience
**Target companies:** 해당 없음. 1순위 독자는 국내주식 개인 투자자.

**Decision-makers:** 개인 투자자 본인. 투자 콘텐츠를 작성할 때는 작성자 본인이 데이터 해석과 발행 여부를 결정한다.

**Primary use case:** 국민연금의 국내주식 5% 이상 보유 종목과 변동 흐름을 빠르게 파악하고, 투자 콘텐츠 작성의 데이터 기반으로 활용한다.

**Jobs to be done:**
- 국민연금 보유 비중 변화와 신규 편입/축소 종목을 확인한다.
- DART 공시 기반 변동 이벤트를 기준일별 포트폴리오 스냅샷으로 정리한다.
- 섹터별 변화와 10% 이상 보유주를 투자 리서치의 출발점으로 삼는다.

**Audience sophistication:** 초보와 중급 사이. 공시, 수급, 섹터 흐름을 완전히 전문적으로 아는 독자보다는, 투자 아이디어를 얻고 싶고 데이터 근거를 이해하고 싶은 개인 투자자.

**Content format:** 블로그 글.

**Content depth:** 기본은 5-7분 분석형 블로그 글. 종목 수가 많거나 섹터 변화가 큰 경우에는 긴 리포트형 글도 사용한다.

**Content structure:** 기본 흐름은 `요약 -> 데이터 표/그림 -> 종목별 코멘트 -> 섹터 해석 -> 투자 유의점 -> 이번 글에서 볼 만한 종목 요약 박스`.

**Visual/content balance:** 표와 그림을 적극적으로 사용한다. 글이 많지 않아도 괜찮으며, 전체 콘텐츠에서 텍스트 비중은 최대 50% 정도로 유지한다. 데이터 표, 차트, 요약 박스가 핵심 정보를 먼저 전달하고 텍스트는 해석을 보완한다.

**Recurring summary box:** 글 마지막에는 "이번 글에서 볼 만한 종목" 같은 요약 박스를 넣는다.

**Preferred headline angles:**
- "국민연금이 최근 늘린 국내주식 5종목"
- "국민연금은 왜 이 섹터를 사고 있을까?"
- "국민연금 10% 이상 보유주 정리"
- "국민연금 매도는 수익실현일까, 비중 조절일까?"

**Use cases:**
- 특정 기준일까지 반영된 국민연금 국내주식 대시보드 생성.
- 5% 이상 보유 공시와 국민연금공단 키워드 기반 공시 수집.
- 포트폴리오, 이벤트 이력, 섹터 변화, 전략 탭을 포함한 통합 HTML 배포.
- CSV 다운로드용 데이터 제공.
- 카카오톡 알림 조건 검토 또는 발송.
- 블로그 글에서 "국민연금이 최근 늘린 종목", "10% 이상 보유 종목", "어느 섹터를 사고 있는지"를 설명.
- 국민연금의 매수가 어떤 투자 논리에서 나왔을 가능성이 있는지 분석.
- 국민연금의 매도가 수익실현인지, 단순 비중 조절인지, 리스크 축소인지 해석.
- 종목별 본문 요약 또는 한 줄 코멘트 작성.

## Problems & Pain Points
**Core problem:** 국민연금 국내주식 보유 변동은 공시가 흩어져 있고 후행적이며, 종목별/섹터별 변화를 누적해서 보려면 수집, 파싱, 정정 반영, 기준일 계산이 필요하다.

**Why alternatives fall short:**
- DART 원문은 정확하지만 투자자가 반복적으로 읽고 비교하기 어렵다.
- 단순 기사나 요약은 원천 이벤트, 기준일, 정정 반영을 체계적으로 확인하기 어렵다.
- 연말 보유 자료만으로는 연중 보유 변동 흐름을 보기 어렵다.

**What it costs them:** 수작업 리서치 시간, 놓친 공시, 기준일 혼동, 종목/섹터 변화 해석 오류.

**Emotional tension:** "국민연금이 샀다/팔았다"는 신호를 보고 싶지만, 공시가 후행적이라 과신하면 안 된다는 불안이 함께 있다. 특히 매도가 나왔을 때 그것이 수익실현인지, 단순 비중 조절인지, 업황 우려인지 구분하기 어렵다.

**Indirect:** 엑셀 수작업 정리, 커뮤니티/블로그 요약, 단순 종목 스크리너.

## Differentiation
**Key differentiators:**
- DART 공시 기반 국민연금공단 보유 변동 이벤트 수집.
- 연말 보유 엑셀과 공시 이벤트를 결합한 기준일별 대시보드.
- 포트폴리오, 섹터 변화, 전략 탭을 통합한 정적 HTML 산출.
- `effective_date`와 `disclosure_date` 기준을 모두 지원.
- CSV 다운로드와 로컬 SQLite 기반 재현 가능성.

**How we do it differently:** 원천 공시 이벤트를 로컬 DB에 저장하고, 기준일별 스냅샷과 섹터 변화 리포트를 정적 산출물로 생성한다.

**Why that's better:** 수작업으로 공시를 뒤지는 시간을 줄이고, 국민연금 보유 흐름을 재현 가능한 데이터로 확인할 수 있다.

**Why customers choose us:** 국민연금의 매수/매도 변화를 단순 나열하지 않고 실적 개선, 저평가, 배당/주주환원, 업황 회복, 섹터 로테이션, 외국인 수급, 주가 과열 여부와 함께 해석할 수 있다.

**Analysis frame for blog content:**
- 실적 개선: 매출, 영업이익, 이익률, 컨센서스 개선 여부.
- 저평가: 업종 내 밸류에이션, 주가 조정 후 가격 매력, PBR/PER 수준.
- 배당/주주환원: 배당 매력, 자사주, 주주환원 정책 변화.
- 업황 회복: 산업 사이클, 수요 회복, 원가 부담 완화, 정책 수혜.
- 섹터 로테이션: 국민연금이 특정 업종 비중을 늘리는지, 줄이는지.
- 외국인 수급: 외국인 순매수 지속 여부와 외국인 지분율 안정성.
- 주가 과열 여부: 60일선 이격, 52주 신고가 이후 거래량 급증, 단기 급등 부담.

**Preferred content output:** 본문 요약과 종목별 한 줄 코멘트. 필요할 때 관심종목, 보류, 제외 같은 판단은 보조적으로 사용한다. 종목별 코멘트는 개인 투자자가 이해하기 쉽게 설명한다.

**Habit:** 기존에는 DART 검색, 뉴스, 엑셀 수작업, 증권 앱 화면을 조합해서 확인한다.

**Anxiety:** 공시의 후행성, 정정 공시 처리, 데이터 누락, 투자 판단으로 오용할 위험.

## Goals
